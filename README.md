# domain-checker 1.0

## `ℹ️` Pierwsze użycie

1. Uzupełnij plik `.env`
2. `npm install`
3. `node index.js`

## `❗` Dodatkowe informacje

- Wymagana jest najnowsza wersja [node.js](https://nodejs.org/)
- Skrypt jest domyślnie ustawiony pod Gmail, jednakże można go skonfigurować pod dowolny serwis webmail
